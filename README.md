# Comunidad de desarrollo de software
CDS es un proyecto construido con [Next.js](https://nextjs.org/)

## Ejecutar en desarrollo 🚀

Para ejecutar la aplicación Next.js necesitara `npm`

```bash
npm run dev
```

### Run Jest tests ✅
```bash
npm run test
```
